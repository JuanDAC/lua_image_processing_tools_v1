operators (tab)
Allows adding metamethods for operators. The first parameter of the metamethod is the object (self) and allows access to private members. __index and __newindex are currently not allowed.

Parameters:

  • tab table A table containing operator function definitions.

Usage:

    operators {
    __add = function(self, other)
    end
}
class (name)
Creates a new class.

Parameters:

  • name string The class name

Returns:

    A function that accepts a table with private, public, static and operator definitions

See also:

  • implements
  • extends
  • static
  • private
  • public
  • operators

Usage:

    class "MyClass" {
   implements {
       "IMyInterface",
       "IAnotherInterface"
   };
   extends "Parent";
   static { ... };
   private { ... };
   public { ... };
   operators { ... };
}
class_body (tab)
Anonymous function returned by class() that accepts a table as a class body. Private, public, static and operator members must be declared in the input table.

Parameters:

  • tab table

Returns:

    the created class table that can be instantiated by calling it (same as import("ClassName")).

See also:

  • class
  • import

is (object, className)
Check if an object is an instance of a class. An object is also considered an instance of its super class, any of its implemented interfaces as well as any interface implemented by the super class.

Parameters:

  • object The object to compare
  • className string the name of the class
